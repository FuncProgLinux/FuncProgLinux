## o/

I'm a SRE and developer from Latin America 👨‍💻. I'm a Free Software Advocate. I trust the work and efforts of GNU and the Free Software Foundation
can lead us to a better, ethical software culture, where sharing knowledge is more important that gatekeeping it for wealth related interests. I plan to keep this mindset more than ever during these times where unjust computing is the norm and even worse, sometimes this type of computing is defended by it's users.

More than ever, In these times I highly encourage you, to research about Free Software, Privacy solutions that your
expertise may craft to help Free Software. You can start here: [Decir no a la informática injusta aunque sea una sola vez es siempre de ayuda.](https://www.gnu.org/philosophy/saying-no-even-once.es.html)

## Projects

Here's a list of projects I'll be actively developing.

### [Urutaú](https://codeberg.org/urutau-ltd/nyctibius) channel

Bits from my private (not proprietary) Guix configuration made public 🗿

Includes some Emacs packages, Go security tools and FreeFem builds. It also contains the
latest MATE Desktop package recipes, and third-party MATE software not available in GUIX.

### [Aile](https://codeberg.org/urutau-ltd/aile)

A small Go library inspired by [hono](https://hono.dev/) and [chi](https://github.com/go-chi/chi), built on top of `net/http` to ease web
server development for both REST/JSON API's and monolith web applications with Go + HTMX.

### [Gavia](https://github.com/urutau-ltd/gavia)

A very eary state work in progress program, very similar to 'My Idlers' for inventory tracking.
